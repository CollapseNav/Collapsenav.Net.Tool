# Collapsenav.Net.Tool

为了自己以后开发方便做了工具包

之前写代码的时候发现有很多实现经常重复的写,比如 `string.IsNullOrEmpty(value)` 这样的

所以就想着稍微封装一下变成 `value.IsEmpty()` `value.NotEmpty()` 这样的调用,大概更加符合"语义"

支持 **netstandard2.0;netstandard2.1;net6**
